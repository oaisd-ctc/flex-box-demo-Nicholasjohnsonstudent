Flex-Direction

This is Row and Row-reverse


These CSS tags move flex items and put them in certain orders.
Row makes it so the flex items are on the left side of the flex box.
Row-reverse makes it so the flex-items move to the left sid of the flex-box and switch from going 1,2,3 to 3,2,1

Flex-wrap

This is flex wrap, no wrap, and wrap-reverse

Wrap makes it so the flex items move one space below if they dont fit on one line.
No wrap makes it so the flex items just resize if they dont fit on that line.
wrap-reverse makes it so when it wraps, the ten goes up one space and counts down
